Typography

The typography used in this website is a cursive style font for the heading and any larger texts, and the body of the text is done in a sans-serif font for easy web reading. The font in the heading goes along well with the logo of the website.

Heading: font-family: 'Baloo', cursive;
Body: font-family: 'Quicksand', sans-serif;
